# AI Course App Prototype

A fully functional UI prototype built with Flutter.

## Features
*   **Interactive Navigation**: Bottom Navigation bar logic implemented.
*   **Login Screen**: Clean authentication UI.
*   **Home Screen**: Highlights the latest AI launches.
*   **Program Listing**: Includes a search UI and list of current courses.
*   **Program Details**: Dynamic screen routing to view specific course data.
*   **Profile/Settings**: View current user progress and logout capabilities.
